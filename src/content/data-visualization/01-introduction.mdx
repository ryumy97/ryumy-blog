# 데이터를 표현하는 100가지 방법 - 소개

데이터를 표현하는 방법은 정말 여러가지 입니다. 이 글들은 저가 프론트 엔드 웹 개발자로서 그 표현방법을 현제 공부하고 있는 내용을 정리하는 블로그 포스트들입니다.

## 왜 데이터 시각화인가?

저는 요즘 취미생활로 러닝을 시작했어요. 현대인들이 러닝을 하면서 필수품으로 가지고 있는 것이 있어요. 스마트 워치.
이 스마트 워치 하나만 있으면 오늘 했던 러닝은 몇 키로미터를 달렸는지, 제 페이스는 어땠는지, 시간은 얼마나 걸렸고, 심박수, 케이던스, 심지어 어떤 코스를 달렸는지 지도로 확인할수도 있죠.

저는 이 많은 데이터를 보고 어떻게 하면 프론트 엔드로 이런 다양한 데이터를 표현하고 또 더 재밌게 보여줄수 있을까 고민하고 있어요.

우리의 생활들은 점점 데이터화 되어가고 있습니다.
우리가 무엇을 먹는지,
우리가 어떻게, 어디로 움직이는지,
우리가 어떻게 입고 활동하고,
어떠한 취미생화을 하는지,
어떠한 패턴으로 생각하고 또 심지어는 어떤것을 좋아할지도 모두 데이터로 저장되고 있습니다.

이것이 제가 느끼는 프론트 엔드 개발자가 데이터를 어떻게 표현할수 있을지, 데이터 시각화를 공부해야 할 이유입니다.

## 시리즈 구성

이 시리즈는 다음과 같이 구성됩니다:

### 1. 기본 요소 (Fundamentals)

- 점(Dots) - 1차원 데이터 표현
- 선(Lines) - 2차원 데이터 표현
- 면(Areas) - 3차원 데이터 표현
- 시간(Time) - 4차원적 데이터 표현
