# 데이터 시각화 개요 - 100가지 방법의 전체 그림

데이터 시각화는 단순히 차트를 그리는 것이 아닙니다. 데이터의 특성, 목적, 대상에 따라 적절한 시각적 표현 방법을 선택하는 것이 핵심입니다.

## 데이터 시각화의 5가지 차원

### 1. 데이터 유형별 분류

#### 정량적 데이터 (Quantitative)

- **연속형**: 온도, 시간, 거리 등
- **이산형**: 인구수, 판매량, 클릭수 등
- **비율형**: 퍼센트, 비율, 확률 등

#### 정성적 데이터 (Qualitative)

- **명목형**: 색상, 브랜드, 카테고리 등
- **순서형**: 등급, 만족도, 중요도 등

### 2. 목적별 분류

#### 탐색적 시각화 (Exploratory)

- 데이터 패턴 발견
- 이상치 탐지
- 관계성 분석

#### 설명적 시각화 (Explanatory)

- 인사이트 전달
- 스토리텔링
- 의사결정 지원

### 3. 복잡도별 분류

#### 단순 시각화

- 막대 차트
- 선 차트
- 파이 차트

#### 복합 시각화

- 대시보드
- 인터랙티브 차트
- 멀티뷰 시각화

## 100가지 방법의 분류 체계

### 기본 요소 (20가지)

1. 점 (Dots) - 개별 데이터 포인트
2. 선 (Lines) - 연결과 흐름
3. 면 (Areas) - 영역과 비율
4. 막대 (Bars) - 비교와 순위
5. 원 (Circles) - 크기와 비율
6. 사각형 (Rectangles) - 구조와 분류
7. 다각형 (Polygons) - 복합 형태
8. 곡선 (Curves) - 부드러운 변화
9. 격자 (Grids) - 체계적 배치
10. 패턴 (Patterns) - 반복과 규칙
11. 색상 (Colors) - 의미와 구분
12. 크기 (Sizes) - 중요도와 비율
13. 투명도 (Opacity) - 깊이와 레이어
14. 회전 (Rotation) - 방향과 각도

- **수치형**: 막대, 선, 산점도

- **시계열**: 선, 영역, 캔들스틱
