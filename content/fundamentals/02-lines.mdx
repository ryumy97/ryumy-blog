# 선(Lines) - 연결과 흐름의 시각화

선은 데이터 시각화에서 연결과 흐름을 표현하는 핵심 요소입니다. 시간에 따른 변화, 관계의 흐름, 그리고 패턴의 연속성을 시각적으로 전달하는 가장 효과적인 방법 중 하나입니다.

## 선의 기본 개념

선은 두 개 이상의 데이터 포인트를 연결하여 관계나 흐름을 나타내는 시각적 요소입니다. 각 선분은 데이터 포인트 간의 관계를 표현하며, 전체적인 선은 데이터의 패턴과 트렌드를 보여줍니다.

### 선의 장점

- **연결성**: 데이터 포인트 간의 관계를 명확하게 표현
- **흐름성**: 시간적 순서나 논리적 흐름을 시각화
- **패턴**: 트렌드와 패턴을 직관적으로 파악 가능
- **효율성**: 많은 데이터 포인트를 간결하게 표현

## 실제 구현 예제

다음은 React와 D3.js를 사용한 선 그래프 예제입니다:

```jsx
import React, { useEffect, useRef } from "react";
import * as d3 from "d3";

const LineChart = ({ data }) => {
  const svgRef = useRef();

  useEffect(() => {
    if (!data || !svgRef.current) return;

    const svg = d3.select(svgRef.current);
    const width = 600;
    const height = 400;
    const margin = { top: 20, right: 20, bottom: 30, left: 40 };

    // Clear previous content
    svg.selectAll("*").remove();

    // Create scales
    const xScale = d3
      .scaleTime()
      .domain(d3.extent(data, (d) => new Date(d.date)))
      .range([margin.left, width - margin.right]);

    const yScale = d3
      .scaleLinear()
      .domain([0, d3.max(data, (d) => d.value)])
      .range([height - margin.bottom, margin.top]);

    // Create line generator
    const line = d3
      .line()
      .x((d) => xScale(new Date(d.date)))
      .y((d) => yScale(d.value))
      .curve(d3.curveMonotoneX);

    // Add line path
    svg
      .append("path")
      .datum(data)
      .attr("fill", "none")
      .attr("stroke", "#3B82F6")
      .attr("stroke-width", 3)
      .attr("d", line);

    // Add dots
    svg
      .selectAll("circle")
      .data(data)
      .enter()
      .append("circle")
      .attr("cx", (d) => xScale(new Date(d.date)))
      .attr("cy", (d) => yScale(d.value))
      .attr("r", 4)
      .attr("fill", "#3B82F6")
      .attr("stroke", "white")
      .attr("stroke-width", 2);

    // Add axes
    svg
      .append("g")
      .attr("transform", `translate(0,${height - margin.bottom})`)
      .call(d3.axisBottom(xScale));

    svg
      .append("g")
      .attr("transform", `translate(${margin.left},0)`)
      .call(d3.axisLeft(yScale));
  }, [data]);

  return (
    <svg
      ref={svgRef}
      width="600"
      height="400"
      style={{ border: "1px solid #ccc" }}
    />
  );
};

export default LineChart;
```

## 선의 다양한 활용

### 1. 시계열 그래프 (Time Series Chart)

시간에 따른 데이터 변화를 보여주는 가장 일반적인 활용입니다.

### 2. 연결 그래프 (Connection Graph)

노드 간의 관계를 선으로 연결하여 네트워크 구조를 표현합니다.

### 3. 스파크라인 (Sparkline)

작은 공간에 트렌드를 간결하게 표현하는 미니어처 선 그래프입니다.

### 4. 영역 차트 (Area Chart)

선 아래 영역을 채워서 양적 변화를 더욱 강조합니다.

## 디자인 고려사항

### 선의 두께

- 데이터의 중요도에 따라 두께 조정
- 너무 얇으면 보이지 않음
- 너무 두꺼우면 시각적 혼잡도 증가

### 선의 색상

- 의미 있는 색상 매핑
- 대비를 고려한 색상 선택
- 접근성을 고려한 색상 조합

### 선의 스타일

- 실선: 연속적인 데이터
- 점선: 불연속적인 데이터
- 대시: 예측이나 추정 데이터

## 마무리

선은 데이터의 연결과 흐름을 표현하는 강력한 시각적 도구입니다. 적절한 디자인과 상호작용을 통해 복잡한 데이터 관계를 직관적으로 이해할 수 있게 해줍니다.
