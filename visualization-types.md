# Data Visualization Types & Examples

This document provides a comprehensive list of data visualization types, from basic charts to advanced interactive visualizations.

## Basic Charts

### 1. Line Charts

- **Simple Line Chart**: Single line showing trends over time
- **Multi-line Chart**: Multiple lines comparing different datasets
- **Area Chart**: Line chart with filled area below the line
- **Stacked Area Chart**: Multiple areas stacked on top of each other
- **Streamgraph**: Flowing, organic area chart
- **Step Chart**: Line chart with horizontal and vertical segments
- **Spline Chart**: Smooth curved line chart

### 2. Bar Charts

- **Vertical Bar Chart**: Bars extending upward
- **Horizontal Bar Chart**: Bars extending to the right
- **Grouped Bar Chart**: Multiple bars grouped together
- **Stacked Bar Chart**: Bars divided into segments
- **100% Stacked Bar Chart**: Bars showing proportions
- **Floating Bar Chart**: Bars with gaps (range charts)
- **Candlestick Chart**: Financial data with open/high/low/close

### 3. Column Charts

- **Simple Column Chart**: Vertical bars
- **Grouped Column Chart**: Multiple columns per category
- **Stacked Column Chart**: Columns divided into segments
- **3D Column Chart**: Three-dimensional columns
- **Cylinder Chart**: Rounded column chart
- **Cone Chart**: Cone-shaped columns
- **Pyramid Chart**: Pyramid-shaped columns

### 4. Pie Charts

- **Simple Pie Chart**: Basic circular chart
- **Donut Chart**: Pie chart with center hole
- **3D Pie Chart**: Three-dimensional pie chart
- **Exploded Pie Chart**: Slices separated from center
- **Nested Pie Chart**: Multiple pie charts within each other
- **Sunburst Chart**: Hierarchical pie chart
- **Polar Area Chart**: Pie chart with varying radii

## Statistical Charts

### 5. Scatter Plots

- **Simple Scatter Plot**: Points showing correlation
- **Bubble Chart**: Scatter plot with varying point sizes
- **3D Scatter Plot**: Three-dimensional scatter plot
- **Scatter Matrix**: Multiple scatter plots in grid
- **Connected Scatter Plot**: Points connected by lines
- **Jitter Plot**: Scatter plot with added noise
- **Hexbin Plot**: Density visualization using hexagons

### 6. Histograms

- **Simple Histogram**: Frequency distribution
- **Stacked Histogram**: Multiple distributions
- **Relative Histogram**: Normalized frequencies
- **Cumulative Histogram**: Running totals
- **2D Histogram**: Two-dimensional frequency distribution
- **Violin Plot**: Combination of box plot and density

### 7. Box Plots

- **Simple Box Plot**: Five-number summary
- **Grouped Box Plot**: Multiple box plots
- **Notched Box Plot**: Box plot with confidence intervals
- **Violin Plot**: Box plot with density curves
- **Bean Plot**: Combination of box plot and density

## Geographic Visualizations

### 8. Maps

- **Choropleth Map**: Areas colored by data values
- **Proportional Symbol Map**: Symbols sized by data
- **Dot Density Map**: Dots representing quantities
- **Flow Map**: Lines showing movement between locations
- **Cartogram**: Distorted map based on data values
- **Heat Map**: Geographic areas with color intensity
- **Isoline Map**: Lines connecting equal values
- **Topographic Map**: 3D terrain visualization

### 9. Contour Plots

- **Simple Contour Plot**: Lines of equal value
- **Filled Contour Plot**: Areas between contour lines
- **3D Contour Plot**: Three-dimensional contour surface
- **Contour Map**: Geographic contour visualization
- **Density Contour**: Statistical density visualization

## Network & Relationship Visualizations

### 10. Network Diagrams

- **Node-Link Diagram**: Nodes connected by edges
- **Force-Directed Graph**: Physics-based layout
- **Hierarchical Tree**: Parent-child relationships
- **Radial Tree**: Tree with circular layout
- **Sankey Diagram**: Flow between categories
- **Alluvial Diagram**: Flow with changing categories
- **Arc Diagram**: Nodes arranged in line with arcs

### 11. Chord Diagrams

- **Simple Chord Diagram**: Circular flow visualization
- **Ribbon Chord Diagram**: Flowing ribbons between segments
- **Interactive Chord Diagram**: Clickable and animated
- **Multi-layer Chord**: Multiple data layers
- **Directed Chord**: Shows direction of flow

## Hierarchical Visualizations

### 12. Tree Maps

- **Simple Treemap**: Nested rectangles
- **Squarified Treemap**: Optimized rectangle shapes
- **Slice-and-Dice Treemap**: Alternating cuts
- **Interactive Treemap**: Clickable and zoomable
- **Circular Treemap**: Circular layout
- **Voronoi Treemap**: Voronoi cell-based layout

### 13. Dendrograms

- **Simple Dendrogram**: Hierarchical clustering
- **Circular Dendrogram**: Circular layout
- **Phylogenic Tree**: Evolutionary relationships
- **Cluster Dendrogram**: Statistical clustering

## Advanced Visualizations

### 14. Heat Maps

- **Simple Heat Map**: Grid with color-coded values
- **Correlation Heat Map**: Correlation matrix
- **Calendar Heat Map**: Time-based heat map
- **Geographic Heat Map**: Location-based heat map
- **Hierarchical Heat Map**: Nested heat maps

### 15. Parallel Coordinates

- **Simple Parallel Coordinates**: Multiple axes
- **Interactive Parallel Coordinates**: Filterable
- **Radial Parallel Coordinates**: Circular layout
- **3D Parallel Coordinates**: Three-dimensional

### 16. Radar Charts

- **Simple Radar Chart**: Multi-dimensional data
- **Filled Radar Chart**: Area-filled radar
- **Multi-series Radar**: Multiple datasets
- **3D Radar Chart**: Three-dimensional radar

## Time Series Visualizations

### 17. Timeline Charts

- **Simple Timeline**: Events on time axis
- **Gantt Chart**: Project timeline
- **Waterfall Chart**: Cumulative changes
- **Candlestick Chart**: Financial time series
- **OHLC Chart**: Open-High-Low-Close data

### 18. Seasonal Plots

- **Seasonal Decomposition**: Trend, seasonal, residual
- **Polar Seasonal Plot**: Circular time representation
- **Calendar Plot**: Calendar-based visualization
- **Cyclical Plot**: Repeating patterns

## 3D Visualizations

### 19. 3D Charts

- **3D Bar Chart**: Three-dimensional bars

- **3D Scatter Plot**: Three-dimensional points
- **3D Line Chart**: Three-dimensional lines

### 20. Volume Visualizations

### 22. Animated Visualizations

## Best Practices

### Design Principles

1. **Clarity**: Make data easy to understand
2. **Accuracy**: Represent data truthfully
3. **Efficiency**: Show maximum information with minimum ink
4. **Aesthetics**: Make visualizations appealing
5. **Accessibility**: Ensure usability for all users

### Color Guidelines

- Use colorblind-friendly palettes
- Maintain consistent color schemes
- Use color to encode data, not decoration
- Consider cultural color associations
- Test with different backgrounds

### Interaction Guidelines

- Provide clear navigation
- Include tooltips and legends
- Allow filtering and zooming
- Maintain performance with large datasets
- Test on different devices
